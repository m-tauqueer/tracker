    Frequently Asked Questions

    Where does the data come from?

    All cryptocurrency data is sourced from the public CoinLore API, which aggregates information from various exchanges and provides real-time market data.

    How often is the data updated?

    The top coins list refreshes automatically every 60 seconds. All other coin details are fetched in real-time when you select a specific cryptocurrency.

    Are the charts real-time?

    The price charts are generated using the current price data and historical patterns. For real-time trading charts, we recommend using dedicated trading platforms.

    Can I track my portfolio?

    Currently, this is a tracking tool for viewing cryptocurrency information. Portfolio tracking features are planned for future updates.

    Is this financial advice?

    No, this tool is for informational purposes only. Always do your own research and consult with financial advisors before making investment decisions.
